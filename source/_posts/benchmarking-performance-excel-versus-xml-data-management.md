## External Diff for XML files

**External Diff With** functionality allows you to compare the content of an XML file from [Files And Folders](https://tools.techidaily.com/advancedinstaller/products/) page with the content of an XML file from disk. 

Advanced Installer automatically detects if one of the following supported tools are installed on the system. :_TortoiseMerge_, _BeyondCompare_,_WinMerge_.

The order from the above list also shows the priority for choosing one tool in the situation when two or more tools are detected on the system.

To perform an XML Diff follow the steps: 

* Select an XML file from [Files And Folders](https://tools.techidaily.com/advancedinstaller/products/) page.
* From the context menu select the “External Diff With” item
* An "Open File" dialog will appear. Select the XML from disk from disk that will be used for the Diff.
* The external Diff tool will be launched

![Important](https://cdn.advancedinstaller.com/svg/common/IconMessageInfo.svg)After the external Diff tool is closed, the XML file contained by the .AIP project will be reloaded.
